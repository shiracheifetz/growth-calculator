### Project Description
The **Investment Growth Calculator** is a web-based financial tool designed to help users estimate the future value of their savings. By leveraging the power of compound interest, the application calculates how an initial investment will grow over time based on a projected annual interest rate and a target retirement year.

---

### Technologies Used
* **HTML5:** Semantic structure.
* **Materialize CSS:** Styling and UI components.
* **JavaScript:** Logic and financial mathematics.
